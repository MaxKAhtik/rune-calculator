# Rune Calculator
Calculates what types of runes are necessary to cast all spells in a user-selected set.

## How to Use
1. Open the plugin panel in the RuneLite sidebar.
2. Check any options that apply to how you would like the rune calculation done.
3. Select spells to include in the rune calculation by clicking on them from the list of spells. You can filter the list by typing spell names in the search bar.
4. Currently selected spells will be shown in compact form underneath the options checkboxes. Click on a spell's icon to remove it from the list, or the red x in the top right to remove all selected spells at once.
5. The runes necessary to cast the selected spells (including options if different types of runes can be used) will be shown underneath that.
