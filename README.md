# Rune Calculator
Calculates what types of runes are necessary to cast all spells in a user-selected set.

## How to Use
1. Open the plugin panel in the RuneLite sidebar.
2. Select spells to include in the rune calculation by clicking on them. Click on the spellbook icons to display spells from different spellbooks.
3. The necessary runes to cast all selected spells will automatically be shown underneath the spell selection interface.
